Лабораторная работа 1: Изучение принципов инкапсуляции — создания класса как единого целого, объединяющего данные и методы для работы с ними. На практике осваивается написание конструкторов (в том числе с параметрами), геттеров/сеттеров и других методов, что является фундаментом для дальнейшего изучения ООП.

Лабораторная работа 2: Исследование механизма наследования, который позволяет создавать новые классы на основе существующих. Это даёт возможность строить древовидные структуры классов (от общего к частному), переиспользуя и расширяя функциональность базовых классов, что делает код более логичным и структурированным.

Лабораторная работа 3: Углубление в ООП через изучение полиморфизма. Создание абстрактных классов и виртуальных методов позволяет определять общий интерфейс для разнородных объектов, а работа с указателями на базовый класс — единообразно управлять ими, что является краеугольным камнем гибкого и масштабируемого дизайна.

Лабораторная работа 4: Освоение шаблонов (template) для написания универсального кода, не зависящего от конкретных типов данных. Изучение и применение на практике основных контейнеров (vector, list, map и т.д.) и алгоритмов (sort, find и т.д.) из STL, что является основой для эффективного программирования на C++.
